JAVASCRIPT

JavaScript is the programming language of the web. It allows me to make websites interactive — from handling form submissions to dynamically updating the page without reloading it.

I’ve used JavaScript extensively in both frontend and backend development. In my E-commerce Website, JavaScript powered everything from adding items to the cart to updating totals in real time. It also handled user login forms, basic validations, and responsiveness alongside React. On the backend, I used JavaScript with Node.js to build APIs that processed orders and managed product data. JavaScript is one of the most versatile tools in my toolbox, and I’m always discovering new libraries and ways to expand its power.
